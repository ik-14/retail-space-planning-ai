## 6. Interaction States

### Hovering over empty shelf slot:
```
  ┌──────────────────────────────────┐
  │ ┌──┬──┬─ ─ ─ ─┬──┬──┐          │
  │ │XX│XX│ :     :│XX│XX│          │  <- Dashed outline = valid drop target
  │ └──┴──┴─ ─ ─ ─┴──┴──┘          │
  └──────────────────────────────────┘
```

### Selected product (in scene):
```
  ┌──────────────────────────────────┐
  │ ┌──┬──┬──┬──┬──┐                │
  │ │XX│##│##│##│XX│                │  <- ## = highlighted/selected (blue outline)
  │ └──┴──┴──┴──┴──┘                │
  └──────────────────────────────────┘
```

### Dragging product from staging:
```
  Scene:
                    ╭──╮
                    │  │  <- Product following pointer (semi-transparent)
                    ╰──╯
  ┌──────────────────────────────────┐
  │ ┌──┬──┬─ ─ ─ ─┬──┬──┐          │
  │ │XX│XX│  SNAP  │XX│XX│          │  <- Nearest valid slot highlights
  │ └──┴──┴─ ─ ─ ─┴──┴──┘          │
  └──────────────────────────────────┘
```
